Cómo montar el servidor en la escuela

  1. Disponer de una PC, notebook o netbook en la institución que pueda destinarse como servidor.
  2. Instalar un sistema operativo GNU/Linux estable (ej. Ubuntu Server o Debian).
  3. Configurar un entorno LAMP (Linux, Apache, MySQL/MariaDB, PHP).
  4. Colocar los recursos navegables en la carpeta /var/www/html/recursos/.
  5. Conectar los equipos de la escuela a la misma red local para acceder al servidor desde cualquier dispositivo.

Servicios disponibles

Moodle

Plataforma educativa local para cursos y seguimiento de estudiantes.

Chamilo

Espacio de red social escolar para compartir ideas y experiencias.

PhET

Colección de simulaciones interactivas para ciencias y matemáticas.

JClic

Actividades educativas interactivas listas para su uso en el aula.

Dokuwiki

Herramienta para documentar y archivar trabajos y proyectos de los estudiantes.

Wikipedia offline

Acceso a CDPedia o Kiwix para consultas sin necesidad de Internet.

OwnCloud

Almacenamiento y compartición de archivos de manera local y segura.

WordPress

Plataforma para la creación de un periódico escolar en red local.

Otros Servicios

Dependiendo de la infraestructura, se pueden montar serrvicios para comunicarse o compartir

Recomendaciones

Asignar un responsable técnico-docente para mantener actualizado el servidor, realizar copias de seguridad y capacitar al equipo en el uso de las plataformas. Promover el uso pedagógico y colaborativo, integrando a toda la comunidad educativa.
